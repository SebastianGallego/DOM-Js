Ejercicios con JavaScript

Calculadora de Aprobación Escolar

Creación de elementos: Crea un script que cree dinámicamente un elemento <div> con un texto dentro y lo agregue al final del cuerpo del documento.


Eliminación de elementos: Crea una lista de elementos y un botón. Al hacer clic en el botón, elimina el primer elemento de la lista.


Cambio de atributos: Crea una imagen en tu página web y utiliza JavaScript para cambiar su atributo src a una imagen diferente.


Cambio de contenido HTML: Crea un párrafo en tu página y utiliza JavaScript para cambiar su contenido HTML a un nuevo párrafo.


Anidación de elementos: Crea un elemento <ul> y agrega varios elementos <li> a él utilizando JavaScript.


Reemplazo de elementos: Crea un elemento <p> en tu página web y utiliza JavaScript para reemplazarlo por un nuevo elemento


Clonación de elementos: Crea un elemento <div> con contenido en tu página y utiliza JavaScript para clonarlo y agregar el clon después del original.


Movimiento de elementos: Crea dos elementos <div> y utiliza JavaScript para mover uno de ellos dentro del otro, convirtiéndolo en un elemento secundario.


Cambio de estilo: Crea un elemento <p> y utiliza JavaScript para cambiar su estilo, como el color de fuente o el tamaño del texto.


Edición de contenido: Crea un elemento <span> con texto en tu página y utiliza JavaScript para permitir que el usuario edite el contenido del <span>
